# Food Recommendation System

A full-stack web application that recommends recipes based on ingredients provided by the user.  
The system uses machine learning concepts to find the best matching recipes and suggests missing ingredients.

---

## Features

- Ingredient-based recipe recommendation  
- Similarity scoring using cosine similarity  
- Displays missing ingredients required for a recipe  
- Step-by-step cooking instructions  
- YouTube video suggestions for recipes  
- Clean and responsive user interface  

---

## Tech Stack

### Frontend
- React (Vite)
- Tailwind CSS

### Backend
- Flask (Python)
- REST API

### Machine Learning
- Cosine Similarity
- Feature Vectorization

---

## How It Works

1. User enters ingredients  
2. Ingredients are converted into vectors  
3. System compares them with dataset recipes  
4. Cosine similarity is used to find best matches  
5. Top recipes are returned with similarity score  
6. Missing ingredients are highlighted
